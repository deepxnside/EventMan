## Getting Started

This project requires Dotnet 6 , you can install Dotnet 6 from [here](https://dotnet.microsoft.com/en-us/download/dotnet/6.0).


1. Clone the repo
   ```sh
   git clone https://github.com/deepxnside/EventMan.git
   ```
     
2. Open the project in any IDE

3. Run the project
  ``` EventMan.Api ```
